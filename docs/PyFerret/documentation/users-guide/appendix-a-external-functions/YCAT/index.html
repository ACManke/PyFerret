YCAT and YCAT_STR

YCAT (A, B )

Concatenates the values of two variables into one list on an abstract Y axis.

Arguments:

A

variables to concatenate in Y
 

B

  

Result Axes:

X

Inherited from variables A and B
 

Y

Abstract, with length the sum of the length of the Y axes of A and B
 

Z

Inherited from variables A and B
 

T

Inherited from variables A and B

Note: This is a grid-changing function. Please see the discussion under the function XCAT about specifying regions.

 

YCAT_STR (sA, sB) concatenates string variables in the Y direction. If YCAT is called with string arguments, Ferret runs YCAT_STR. The two arguments to YCAT must be of the same type.

 

yes? let a = ysequence({"Q", "R", "S"})
yes? let b = ysequence({"U", "V", "W"})
yes? list ycat(a,b)
