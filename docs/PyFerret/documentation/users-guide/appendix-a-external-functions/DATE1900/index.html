DATE1900

A DATE1900(formatted date) Scalar function: converts a formatted date into Julian days since 1-Jan-1900.

Argument:

formatted date

dd-mmm-yyyy or dd-mmm-yyyy, in quotes
     

Result Axes:

X

NORMAL (no axis)
 

Y

NORMAL (no axis)
 

Z

NORMAL (no axis)
 

T

NORMAL (no axis)

Examples: 

yes? list date1900("23-feb-2003")
VARIABLE : DATE1900("23-feb-2003")
37673.
yes? list date1900("2-jan-1900")
VARIABLE : DATE1900("2-jan-1900")
1.000
