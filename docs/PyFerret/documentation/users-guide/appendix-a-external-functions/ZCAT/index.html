ZCAT and ZCAT_STR

ZCAT (A, B )

Concatenates the values of two variables into one list on an abstract Z axis.

Arguments:

A

variables to concatenate in Z
 

B

  

Result Axes:

X

Inherited from variables A and B
 

Y

Inherited from variables A and B
 

Z

Abstract, with length the sum of the length of the Z axes of A and B
 

T

Inherited from variables A and B

Note: This is a grid-changing function. Please see the discussion under the function XCAT about specifying regions.

 

ZCAT_STR (sA, sB) concatenates string variables in the Z direction. If ZCAT is called with string arguments, Ferret runs ZCAT_STR. The two arguments to ZCAT must be of the same type.

 

yes? let a = zsequence({"Q", "R", "S"})
yes? let b = zsequence({"U", "V", "W"})
yes? list zcat(a,b)
